# Global Temperature Visualization Dashboard

An exploratory data analysis and interactive dashboard project that analyzes long-term global temperature anomaly data — visualizing trends, detecting outliers, testing statistical significance, and presenting it all through a live **Streamlit** dashboard.

## Overview

The dataset tracks global temperature anomalies (deviation from a historical baseline) from **1850 onward**, broken down by month and year, along with smoothed 5-year, 10-year, and 20-year rolling anomaly values and their associated uncertainty measurements.

This project walks through a full analysis pipeline:

1. **Data loading & inspection**
2. **Visualization** — trend lines, comparative plots, and scatter plots
3. **Statistical analysis** — correlation, moving averages, anomaly detection, trend fitting, and hypothesis testing
4. **Interactive dashboard** — a Streamlit app for exploring the data by year range

## What's Inside

### Exploratory Visualizations
- Annual and monthly temperature anomaly trends over time
- Five-year vs. ten-year anomaly comparison
- Monthly anomaly vs. uncertainty scatter plot
- Combined view of annual, 5-year, 10-year, and 20-year anomalies

### Statistical Analysis
- **Correlation matrix** — relationships between anomaly metrics at different time scales
- **Moving averages** — 5-year and 10-year rolling means to smooth short-term noise and reveal long-term trends
- **Anomaly/outlier detection** — z-score-based identification of unusually warm or cool years
- **Trend analysis** — linear regression fit to quantify the rate of warming over time
- **Hypothesis testing** — a two-sample t-test comparing anomalies before vs. after 1950, to check whether the warming trend is statistically significant rather than random fluctuation

### Interactive Dashboard
A Streamlit app (`app.py`) that lets you:
- Filter the dataset by a custom year range using a slider
- View the filtered data as a table
- See an interactive line chart of annual anomalies over the selected range
- See a histogram showing the distribution of annual anomalies

## Key Findings

- Global temperature anomalies show a clear long-term **upward trend**, consistent with global warming.
- The difference in anomalies before vs. after 1950 is **statistically significant** (very low p-value), indicating the shift is not due to random variation.
- Moving averages confirm the trend isn't driven by short-term volatility — warming holds up even after smoothing.

## Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/<your-username>/global-temperature-dashboard.git
cd global-temperature-dashboard
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Add the dataset
Place `Global Temperature.csv` in the `data/` folder, then update the file path in the notebook/app from the Colab path (`/content/Global Temperature.csv`) to your local path (e.g. `data/Global Temperature.csv`).

### 4. Run the notebook
Open `Global_Temperature_Visualization_Dashboard.ipynb` in Jupyter to step through the EDA and statistical analysis.

### 5. Run the dashboard locally
```bash
streamlit run app.py
```
This opens the dashboard directly in your browser at `http://localhost:8501` — no ngrok tunnel needed outside of Colab.
